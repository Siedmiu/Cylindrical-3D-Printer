# FullcontrollXYZ Preview

Preview of a forked [FullControl XYZ](https://github.com/Siedmiu/fullcontrol_Cylindrical) adapted to generate toolpaths for a cylindrical 3D printer.

> **Work in progress.** This path-generation method is experimental. Using a slicer recalculation method is recommended over this approach.

## Requirements

The notebooks require:
- The [FullControl XYZ](https://github.com/fullcontrolxyz/fullcontrol) package
- Additional functions for cylindrical printers
All of this is available in the [fork repository](https://github.com/Siedmiu/fullcontrol_Cylindrical)

## Contents

- [cylindrical_printer_tutorial.ipynb](cylindrical_printer_tutorial.ipynb) — tutorial notebook walking through cylindrical path generation
- [cylindrical_printer_tutorial.md](cylindrical_printer_tutorial.md) — rendered preview of tutorial results
- [cylindrical_slicer.ipynb](cylindrical_slicer.ipynb) — slicer notebook; best used with simpler STL files as it is prone to errors and can take a long time to run
